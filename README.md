# Collage Maker — онлайн-сервис создания фотоколлажей

Быстрый браузерный инструмент для создания красивых фотоколлажей с **прозрачным фоном между фотографиями** — без регистрации, без серверов, полностью на клиенте.

**Демо:** [безлишних.рф](https://безлишних.рф/)

---

## Главное преимущество

Основная фишка сервиса — **поддержка прозрачного фона** (PNG с альфа-каналом). Это позволяет создавать коллажи с прозрачными зазорами между фото, которые можно вставлять на любой фон в других редакторах без белой или чёрной рамки.

## Возможности

- **16 готовых макетов** — от одного фото до сеток 3×2 и 2×3
- **Прозрачный фон** — экспорт PNG без фона
- **Настраиваемая рамка** — ширина (0–50px), цвет, отступы
- **Скругление углов** — радиус 0–100px для каждой ячейки
- **Drag & Drop** — перетащите фото прямо в ячейки или в панель загрузки
- **Перемещение фото внутри ячейки** — позиционирование и масштаб
- **Экспорт PNG / JPG** — с выбором качества и множителем ×2 / ×3
- **Пресеты размеров** — 1:1, 4:3, 9:16, 16:9
- **Работает офлайн** — никаких серверных запросов, все данные остаются в браузере

## Быстрый старт

```bash
git clone https://github.com/gafar87/collage.git
cd collage/collage-app
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173)

## Сборка для продакшена

```bash
npm run build
```

Статические файлы появятся в `collage-app/dist/` — можно деплоить на любой хостинг (Nginx, GitHub Pages, Netlify и т.д.).

## Как пользоваться

1. **Загрузите фото** — перетащите в нижнюю панель или нажмите на зону загрузки
2. **Выберите макет** — в левом сайдбаре
3. **Перетащите фото в ячейки** — или кликните на ячейку и выберите фото
4. **Настройте внешний вид** — фон, рамка, скругление
5. **Скачайте результат** — кнопка «Скачать» в шапке, PNG или JPG
