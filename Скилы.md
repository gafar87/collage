landing-page-copywriter — опытный рерайтер лендингов: переписывает описания функций, заголовки, CTA и секции по фреймворкам PAS, AIDA, StoryBrand для повышения конверсии

### Как пользоваться landing-page-copywriter

**Вызов:** `/landing-page-copywriter`

**Что умеет:**
- Пишет и переписывает копирайт лендинга по проверенным фреймворкам (PAS, AIDA, StoryBrand)
- Создаёт заголовки (hero headline), подзаголовки, CTA, описания функций
- Формирует секции: проблема → решение → как работает → социальные доказательства → FAQ → финальный CTA
- Оптимизирует текст под конверсию, а не просто «красивость»

**Перед запуском скажи ему:**
1. Что за продукт / сервис
2. Целевая аудитория (кто покупатель)
3. Какую проблему решает
4. Чем отличается от конкурентов
5. Какое целевое действие (кнопка «Купить», «Оставить заявку» и т.д.)
6. Есть ли цифры / отзывы / кейсы для социального доказательства

**Примеры запросов:**
```
/landing-page-copywriter
Перепиши секцию "Возможности" для нашего сервиса тендерной аналитики.
Аудитория — закупщики B2B. Проблема — теряют выгодные тендеры из-за медленного мониторинга.
```
```
/landing-page-copywriter
Напиши hero-секцию по фреймворку AIDA для SaaS-платформы автоматизации отчётов.
```

**Установка:** `npx skills add onewave-ai/claude-skills@landing-page-copywriter -g -y`

### Как пользоваться review-and-refactor

**Вызов:** /review-and-refactor

**Что умеет:**
- Читает гайдлайны проекта (`.github/instructions/*.md`, `.github/copilot-instructions.md`) и рефакторит код под них
- Улучшает читаемость, убирает дублирование, приводит к единому стилю
- Не ломает логику и не меняет API/интерфейсы
- Следит чтобы тесты продолжали проходить

**Ограничение:** не разбивает файлы на части — только очищает код внутри существующих файлов.
Для разбивки на файлы (вынос JS/CSS) — просить напрямую, без скилла.

**Примеры запросов:**
```
/review-and-refactor
Почисти static/index.js — убери дублирование, приведи к единому стилю.
```
```
/review-and-refactor
Рефактори passport_check.py по принципу единой ответственности.
```
